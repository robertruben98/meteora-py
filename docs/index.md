# meteora-py

Python client for the Meteora DLMM Data API — Solana liquidity pools, metrics, OHLCV. Sync + async, Pydantic v2.

## Install

```bash
pip install meteora-py
```

The distribution is `meteora-py`; the import package is `meteora`.

## Quick start

```python
import meteora
```

See the [API reference](api.md) for the full surface.
